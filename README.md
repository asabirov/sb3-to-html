# Headless converter sb3 (Scratch v.3) to HTML

## How to use

      npm install
      npm i -g ts-node
      bin/sb3-to-html examples/example_1.sb3 output/

## Make a symlink

      sudo ln -s bin/sb3-to-html /usr/local/bin/sb3-to-html

## Updating turbowarp

      bash /bin/download_turbowarp.sh 0.2.0
